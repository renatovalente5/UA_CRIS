Develop a CRIS for UA for publications

Objective: Complete RIA with its missing publications

Publication source: ORCID


T 1 : Obtain the difference between ORCID and RIA

        T 1.1 : Obtain a list of UU/IUPI from all the UA's researchers in RIA

        T 1.2 : Convert IUPI in ORCID ID using UA's ID

        T 1.3 : Obtain a list of DOIs of the publications from researchers identified in T1.1

        T 1.4 : Obtain a list of DOIs of the publications through researchers ORCID ID via PTCRISync

        T 1.5 : Obtain the list of missing publications (difference)


T 2 : Transition of the missing publications to RIA with a verification and addition of preprint, embargo policies and quality control

        T 2.1 : Establish of the login interface

        T 2.2 : Establish of the home screen

        T 2.3 : Interface to select which publications will be imported from ORCID to RIA

                T 2.3.1 : Establish the user interface

                T 2.3.2 : Establish the data structure that will store the chosen publications

                T 2.3.3 : Develop the importation functionality

                T 2.3.4 : Create a state for the publications

        T 2.4 : Verification, addition of preprint, establishment of the embargo policy

                T 2.4.1 : Establish the fields that will be edited

                T 2.4.2 : Establish the user interface

                T 2.4.3 : Develop the functionality


T 3 : Architecture

        T 3.1 : Choice of the software library from which the user interface will be developed

        T 3.2 : Internal architecture
